# Gerbtrace Docs

Use this documentation as the single source for expected behavior and operator workflows.

## Primary workflows

- Import customer Gerber, drill, BOM, and PnP files in the `Files` tab.
- Resolve file typing and field mappings if auto-detection is incomplete.
- Validate PCB dimensions and panel setup before export.
- Review BOM and PnP mismatches before handing over to production.
